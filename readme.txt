Easy to use Overlay popus gallery in WordPress.Base on jQuery Library.

== Description ==

 Features:    
1. In addition to the jQuery library, only need to load some little JS files and some little of CSS files.  
2. The output code of the plug-in on the page in line with W3C's XHTML 1.1 and CSS 3 standards, and the code structure is very simple.   
3. When deactivated，the images calling code generated by the plug-in will not cause any adverse effects.

This plugin allows you easily to make [Overlay gallery](http://flowplayer.org/tools/demos/overlay/gallery.html) popups on WordPress.

这个插件助你在 WordPress 中轻易地使用 [Overlay gallery](http://flowplayer.org/tools/demos/overlay/gallery.html) 相册功能。

= Supported Languages: =

== Installation ==

1. Unzip archive to the '/wp-content/plugins/' directory.
    解压文件到“/wp-content/plugins/”目录。
2. Activate the plugin through the 'Plugins' menu in WordPress.
     在WordPress后台“插件”页面激活Overlay4WP。

■Custom style:
    自定义样式：
    Create a CSS file named "overlay4wp.css" under current theme directory.
    在当前主题目录建立名为“overlay4wp.css”的样式表文件。
